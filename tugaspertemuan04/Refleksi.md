Java menyediakan lima operator aritmatika dasar untuk manipulasi data numerik, yaitu penjumlahan (+), pengurangan (-), perkalian (\*), pembagian (/), dan modulo (%) seperti bahasa pemrograman umum lainya (c++, py, dan lainnya).
